SQL Analysis
Funnel Drop-off Query — BigQuery / GA4 Export
SQL used to calculate stage-by-stage conversion rates from GA4 raw event export.
-- Used Car Buyer Funnel: Stage-by-Stage Drop-off Analysis
-- Source: GA4 BigQuery Export · events_* table · Date: 2024-Q4

WITH funnel_stages AS (
  SELECT
    user_pseudo_id,
    MAX(CASE WHEN event_name = 'page_view'
      AND page_location LIKE '%/home%'       THEN 1 ELSE 0 END) AS stage_landing,
    MAX(CASE WHEN event_name = 'search_submit'               THEN 1 ELSE 0 END) AS stage_search,
    MAX(CASE WHEN event_name = 'listing_view'                THEN 1 ELSE 0 END) AS stage_listing,
    MAX(CASE WHEN event_name = 'vhr_page_view'               THEN 1 ELSE 0 END) AS stage_vhr,
    MAX(CASE WHEN event_name = 'purchase'                    THEN 1 ELSE 0 END) AS stage_purchase
  FROM `carfax_analytics.events_*`
  WHERE _TABLE_SUFFIX BETWEEN '20241001' AND '20241231'
  GROUP BY user_pseudo_id
)
SELECT
  COUNTIF(stage_landing  = 1)                              AS landing_sessions,
  COUNTIF(stage_search   = 1)                              AS search_sessions,
  COUNTIF(stage_listing  = 1)                              AS listing_sessions,
  COUNTIF(stage_vhr      = 1)                              AS vhr_sessions,
  COUNTIF(stage_purchase = 1)                              AS conversions,
  ROUND(COUNTIF(stage_purchase=1) / COUNTIF(stage_landing=1) * 100, 2) AS overall_cvr_pct
FROM funnel_stages;
Device Segmentation
Mobile vs Desktop CVR Gap
Mobile users convert at 34% lower rate — significant UX opportunity.
Key Findings
68%
VHR Page Bounce Rate
Biggest single drop-off point. Users reach the VHR page but don't purchase. Root cause: price anchoring shock ($39.99) without sufficient trust signals or value explanation above the fold.
34%
Mobile CVR Gap vs Desktop
Mobile sessions represent 61% of traffic but only 38% of conversions. Checkout flow on mobile has 3 unnecessary steps vs desktop — friction is measurable and fixable.
9.8%
Email Channel CVR — Highest
Email-sourced users convert at 4.7× the rate of social media users. Lifecycle email sequences targeting saved searches and price drop alerts show the strongest ROI signal.
Recommendations
REC — 01
VHR Page Value Redesign
Add trust signals (sample report preview, # of accidents found stat, money-back guarantee) above the fold. A/B test price framing: "$39.99 one-time" vs "Less than 0.1% of the car's value."
↑ Est. +1.8pp CVR · $28K revenue lift
REC — 02
Mobile Checkout Streamline
Reduce mobile checkout from 5 steps to 2 using Apple Pay / Google Pay as primary CTA. Remove mandatory account creation — offer guest checkout with email capture post-purchase.
↑ Est. +2.1pp mobile CVR · $10K lift
REC — 03
Email Lifecycle Expansion
Email converts at 9.8% — highest of all channels. Build triggered sequences: saved search alerts, price drop notifications, "X people viewed this car" urgency signals for high-intent users.
↑ Est. +3.2pp email CVR · $18K lift
REC — 04
Paid Search Audience Refocus
Paid search CVR (4.2%) is half of organic (8.1%). Current keyword targeting is too broad — shift budget to high-intent terms ("VIN check", "car history report") and exclude informational queries.
↓ Est. -22% CPA · $8K budget saved
